Access to electronic filing formalities

To access the electronic filing formalities on the LBR website, you need to have a "Luxtrust certificate" user account.

Only a person authenticated by the LBR with a digital Luxtrust certificate has the right to electronically file documents with the RCS.

If you already have a Luxtrust certificate, the process for creating your user account is automatic and your user account can be used immediately. No access request form needs to be completed in order to create your user account.

How to create a new "Luxtrust certificate" user account?

To create a new account just click on "Connect" in the toolbar and follow the instructions on the screen.

You first have to click on the link "Click here to register" and then on "Or click here to register with your electronic Luxtrust certificate" and follow the instructions on the screen.

For more information on how to create a "Luxtrust certificate" account, please consult the "Connection process" document.

Have you already created a user account with a “Luxtrust certificate”?

If you already have a user account associated with your “Luxtrust certificate” on the LBR website, you should connect and use this account to access the electronic filing menu.
